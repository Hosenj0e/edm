# Offline EDM - Система управления документами

Офлайн система электронного документооборота для учебного заведения с поддержкой синхронизации с порталом НовГУ.

## Возможности

- 📋 Управление учебными группами и студентами
- 📝 Создание и подписание ведомостей
- 🔐 Шифрование данных и цифровые подписи
- 🌐 Синхронизация с порталом НовГУ
- 📥 Импорт групп из HTML файлов
- 🔍 Автопоиск групп по номерам
- 📊 Экспорт ведомостей в HTML

## Сборка

### Требования

- CMake 3.16+
- Qt 6.5+
- C++17 компилятор (MSVC 2019+, GCC 9+, Clang 10+)
- OpenSSL (для шифрования)

### Компиляция

```bash
# Создание build директории
cmake -S . -B build

# Сборка Debug
cmake --build build --config Debug

# Сборка Release
cmake --build build --config Release
```

### Запуск

```bash
# Windows
build\Debug\offline-edm.exe

# Linux/macOS
./build/offline-edm
```

## Первый запуск

При первом запуске создайте администратора:
- **Логин**: admin
- **Пароль**: 123 (измените после входа)

## Импорт групп

### Способ 1: HTML импорт (рекомендуется)

1. Откройте страницу группы на portal.novsu.ru в браузере
2. Нажмите Ctrl+S и сохраните HTML
3. В приложении: "⟳ Импорт групп" → "Открыть папку"
4. Скопируйте HTML файлы в папку
5. "Импортировать сейчас"

### Способ 2: Автопоиск групп

1. Войдите один раз с логином/паролем и включите "Запомнить сессию"
2. "⟳ Импорт групп" → "🔍 Автоматический поиск групп"
3. Введите паттерн:
   - `2991` - конкретная группа
   - `29*` - все группы 2900-2999
   - `2900-2950` - диапазон
4. "🔍 Поиск по паттернам"

### Способ 3: Онлайн синхронизация

1. "⟳ Импорт групп" → "Онлайн синхронизация"
2. Введите URL страницы со списком ваших групп
3. Логин/пароль от портала
4. "Войти и загрузить группы"

## Тестирование

```bash
# Сборка с тестами (по умолчанию включено)
cmake -S . -B build -DOFFLINE_EDM_BUILD_TESTS=ON
cmake --build build --config Debug

# Запуск тестов
ctest --test-dir build --output-on-failure
```

## Отладка

Для включения отладочного вывода Qt:

```bash
# Windows CMD
set QT_LOGGING_RULES=*.debug=true
build\Debug\offline-edm.exe

# Windows PowerShell
$env:QT_LOGGING_RULES='*.debug=true'
.\build\Debug\offline-edm.exe

# Linux/macOS
QT_LOGGING_RULES='*.debug=true' ./build/offline-edm
```
