# Система электронного документооборота (Дипломный проект)

Десктопное приложение для управления академическими ведомостями с поддержкой импорта данных из портала НовГУ.

**Автор:** Пежемский Артур  
**Группа:** 2991  
**Университет:** НовГУ им. Ярослава Мудрого

---

## О проекте

Система позволяет преподавателям:
- Управлять учебными группами и студентами
- Создавать экзаменационные и зачетные ведомости
- Импортировать списки студентов из портала НовГУ
- Работать офлайн с локальной базой данных
- Экспортировать ведомости для печати

### Используемые технологии

- **Язык:** C++17
- **Фреймворк:** Qt 6.5 (QML для интерфейса)
- **База данных:** SQLite3
- **Сборка:** CMake
- **Безопасность:** OpenSSL для шифрования

## Быстрый старт

### Установка зависимостей

**Windows:**
- Установите Qt 6.5+ с официального сайта
- CMake 3.16+
- Visual Studio 2019 или новее

**Linux:**
```bash
sudo apt install cmake qt6-base-dev qt6-declarative-dev libssl-dev
```

### Сборка проекта

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Hosenj0e/edm.git
cd edm

# 2. Создать папку для сборки
cmake -S . -B build

# 3. Скомпилировать
cmake --build build --config Debug

# 4. Запустить
build\Debug\offline-edm.exe    # Windows
./build/offline-edm             # Linux
```

### Первый запуск

При первом запуске войдите с учетными данными по умолчанию:
- **Логин:** test
- **Пароль:** 123

## Основные возможности

### 1. Управление группами
- Создание новых учебных групп
- Добавление студентов вручную
- Просмотр информации о группах

### 2. Импорт из портала НовГУ

**Способ А: Импорт из HTML**
1. Откройте страницу группы на portal.novsu.ru
2. Сохраните страницу (Ctrl+S)
3. В приложении: "⟳ Импорт" → Выберите HTML файл

**Способ Б: Автопоиск**
1. Введите номер группы или шаблон (например: `2991` или `29*`)
2. Система автоматически найдет и загрузит данные

### 3. Работа с ведомостями
- Выберите группу и дисциплину
- Создайте новую ведомость (экзамен/зачет)
- Заполните оценки
- Экспортируйте для печати
