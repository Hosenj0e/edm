# Автоматический поиск групп на портале

## Проблема

На портале `https://portal.novsu.ru/` список всех групп скрыт и доступен только через поиск. Для преподавателей доступны только их собственные группы.

## Решение: Автоматический перебор номеров групп

Система может автоматически проверять существование групп по заданным паттернам номеров.

## Как использовать

### Способ 1: Автопоиск с паттернами по умолчанию

```cpp
appController.autoSearchGroups()
```

**Что ищется:**
- Группы текущего и прошлых 3 лет
- Форматы: 2401, 2402... 2499, 2301... 2399 и т.д.
- Примерно 300-400 номеров

### Способ 2: Поиск конкретных групп

```cpp
appController.autoSearchGroups(["2991", "2992", "3001"])
```

### Способ 3: Поиск по диапазону

```cpp
appController.autoSearchGroups(["2400-2450", "3000-3050"])
```

### Способ 4: Поиск по маске

```cpp
appController.autoSearchGroups(["24*", "30*"])
// Ищет: 2400-2499 и 3000-3099
```

## Как это работает

1. **Генерация списка номеров** - по паттернам создается список возможных номеров групп
2. **Последовательная проверка** - для каждого номера делается запрос на портал: 
   `https://portal.novsu.ru/group?name=НОМЕР`
3. **Парсинг результатов** - если группа существует, она парсится и добавляется в БД
4. **Пропуск несуществующих** - если группа не найдена, переходим к следующему номеру

## Ограничения

- **Скорость**: ~10 групп в секунду (с учетом задержек между запросами)
- **Нагрузка на портал**: используйте разумные паттерны, избегайте проверки тысяч номеров
- **Сессия**: требуется активная сессия портала (вход с логином/паролем)

## Рекомендации

1. **Для поиска всех групп учебного заведения:**
   - Уточните формат номеров групп (у каждого вуза свой)
   - Используйте диапазоны или маски для ограничения поиска

2. **Для периодического обновления:**
   - Сохраняйте список найденных номеров
   - Проверяйте только новые возможные номера

3. **Оптимизация:**
   - Группы обычно имеют логичную нумерацию (по годам, специальностям)
   - Начните с известных паттернов вашего учебного заведения

## Пример использования в QML

```qml
Button {
    text: "Автопоиск групп 2024 года"
    onClicked: {
        appController.autoSearchGroups(["24*"])
    }
}

Button {
    text: "Найти все группы (осторожно!)"
    onClicked: {
        // Проверит ~300 номеров, займет ~30 секунд
        appController.autoSearchGroups()
    }
}
```

## Альтернативы

Если автопоиск не подходит:

1. **Ручное сохранение HTML** - откройте нужные группы в браузере и сохраните HTML
2. **Список от администрации** - получите официальный список номеров групп
3. **Парсинг из личного кабинета** - если есть доступ к полному списку

## URL для поиска на портале

- Список групп: `https://portal.novsu.ru/search/groups/`
- Конкретная группа: `https://portal.novsu.ru/group?name=НОМЕР`
- Ваши группы: `https://portal.novsu.ru/teacher/groups` (для преподавателей)
